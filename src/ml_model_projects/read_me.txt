libraries to install using pip

COre utilities:
1. pip install certifi
2. pip install python-dotenv


Langchain:
1. pip install langchain-community
2. pip install langchain-text-splitters
3. pip install langchain

Vector Store (FAISS):
1. pip install faiss-cpu

Embeddings (HuggingFace):
1. pip install sentence-transformers
2. pip install transformers
3. pip install torch

PDF Loading:
1. pip install langchain-community

Local LLM Integrations:
1. pip install ollama

Can use following to install all:
pip install certifi python-dotenv ollama \
langchain langchain-community langchain-text-splitters \
faiss-cpu sentence-transformers transformers torch
